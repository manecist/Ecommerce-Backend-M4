SUBIDA DE ARCHIVOS DE LA CONSOLA JAVA
SE CREA PACKAGE DE ECOMMERCE CON PRODUCTOS, CARRITO Y TIENDA

* PRODUCTOS TIENE CONSIGO OTROS PACKAGE DE INTERFACES, IMPLEMENTACION Y ENUNCIADOS

-> ENUNCIADOS TIENE CONSIGO ENUM CATEGORIA Y SUBCATEGORIA

-> INTERFACES TIENE INTERFACE IPRODUCTO, IPRODUCTOREPOSITORY, IREGLADESCUENTO

-> PACKAGE DE IMPLEMENTACION TIENE CONSIGO DESCUENTOS Y REPOSITORIO, ADEMAS DE UNA CLASE DE PRODUCTO

--> DESCUENTOS TIENE CLASES DE CATEGORIA, SUBCATEGORIA, POR MONTO Y MANAGER

--> REPOSITORIO TIENE UNA UNICA CLASE LLAMADA PRODUCTOREPOSITORY

** LOS ENUM ESTAN UTILIZADOS DENTRO DE LA CLASE PRODUCTO QUIEN IMPLEMENTA LA INTERFAZ IPRODUCTO
SE CREARON LOS ENUM PARA HACER LA PAGINA MAS MANTENIBLE AL MOMENTO DE DESEAR CREAR OTRA CATEGORIA O SUBCATEGORIA EXTRA SIMPLEMENTE SE AGREGA DENTRO DE LOS ENUM
